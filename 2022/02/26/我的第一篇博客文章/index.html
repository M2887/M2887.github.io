记录建立博客过程中遇到的问题

第一次利用Hexo搭建自己的静态博客，遇到如下问题，

  • 博客建立的步骤是
    hexo clean
    hexo g
    hexo d
    清理-生成-提交
  • VScode使用还不太熟练，导致_config.yml中被我误输入了好几个字符，导致无法正常提交
  • 网络问题
    服务器的网络无法连接到GIT
    解决办法，百度后，在服务器输入如下指令
    git config –global –unset http.proxy
    git config –global –unset https.proxy
    —此方法无效，淦
